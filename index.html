The translation toolkit is a set of recommendations for translation practices to be used by researchers, librarians or cultural heritage workers. In our first iteration we offer some advice on translation for conferences. In the future we hope to offer some advice for translating software and articles.

If you would like to contribute send us a line.

{% else if site.lang == "es" %}

La caja de herramientas de traducción es una serie de consejos prácticos para el trabajo de traducción acarreado por académicos, bibliotecarios y demás trabajadores del sector cultural. En esta primera versión de la caja nos enfocamos en la traducción durante conferencias. En el futuro esperamos agregar recomendaciones para la traducción de páginas web y artículos.

Si desea contribuir envienos un mensaje por correo electrónico.
